Zemheri Resort

This Django project is a personal diary web application designed to allow users to securely record their daily thoughts and experiences. Users can create an account, log in, and write, edit, or delete diary entries. With a clean, intuitive interface and robust security features, this diary application ensures that users' personal entries are kept private and accessible only to them. Built with Django, the project emphasizes user experience, data integrity, and privacy.

Frontend : HTML , CSS , Bootstrap

Backend : Django

Rest Api : Django Rest Framework

Database : SQLite
