Zemheri Cars

This Django project is a comprehensive web application for a second-hand used car shop, facilitating the buying pre-owned vehicles. The platform includes features such as user registration, car listings with detailed descriptions and images and an integrated contact form for inquiries. The admin panel allows for easy management of car listings and transactions. Built with Django, this project ensures a secure, scalable, and user-friendly experience for car buyers.

Frontend : HTML , CSS , Bootstrap

Backend : Django

Database : MySQL
